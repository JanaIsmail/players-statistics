This is a java spring boot application that retrieves statistics for tennis players.

To run the application locally you need to have maven installed and added to your environment variables.
You can then navigate to the root of the project from the command line and execute mvn spring-boot:run.
From your browser go to http://localhost:8080 and call the endpoints (API provided in players-api.yaml)

The application is also deployed on https://players-statistics-application.onrender.com where you can navigate and call the endpoints.
